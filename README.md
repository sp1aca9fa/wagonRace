# wagonRace

Built as part of a Le Wagon bootcamp exercise.

A two-player keyboard racing game set against a Tokyo backdrop. Each player mashes their key to advance their car across a 50-tile track — first to the finish line wins.

## Gameplay

| Player | Key | Car color |
|--------|-----|-----------|
| Player 1 | `p` | Red |
| Player 2 | `q` | Yellow |

Press your key repeatedly to move your car forward. The first player to reach the finish tile wins. Reload the page (`F5`) to restart.

## Tech stack

- HTML5 / CSS3
- Vanilla JavaScript (ES modules)
- Bootstrap 5

## Setup

```bash
git clone https://github.com/sp1aca9fa/wagonRace.git
cd wagonRace
npx serve
```

Open in your browser: http://localhost:3000
